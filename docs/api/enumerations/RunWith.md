# Enumeration: RunWith

The different ways that we run the test files

## Enumeration Members

### Node

> **Node**: `"node"`

This only works with .js files or files that are transformed to a js file first

***

### TsNode

> **TsNode**: `"ts-node"`

This will run ts-node [script.ts]

***

### Tsx

> **Tsx**: `"tsx"`

This will run via tsx [script.ts]
